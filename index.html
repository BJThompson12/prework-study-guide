CSS

  • Cascading Style Sheets (CSS) is a programming language used by developers to define how webpage content is presented to users. This includes the colors, aesthetucs, visual layout, and organization.CSS is used to define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • Inline CSS: Used to style an element directly in the HTML file. Applies to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only. Quick and permanent changes but are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change.
  • Internal CSS style sheet: Embed an entire stylesheet directly in the HTML by using style element in the head element. Rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
  • External CSS style sheet: Allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • Selector: define the element or attributes to which the rules, or declarations will apply
  • Declarations contain: CSS property you want to apply and the value of the property.
  • Margin property indicates how much space we want around the outside of an element. The margin is not visible on the page
  • Padding property adds space around the content inside an element. When you increase padding, you are actually making the entire element bigger.
  • Wild card selector allows you to apply the rule to all the elements visible on the page
  • You can combine 2 or more selectors with a comma
  • Block value assigns to start on a new line
  • Link element creates a connetion to an external file
  • Rel attribute specifies the location (URL) of the external resource
  • href uses the relative path
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector. Then the class attribute can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.
  • Box Shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color. The X offset defines the width or extent of the shadow on the X axis (or to the right of the box); the Y offset defines the extent on the Y axis (or just below the box)

  • Separation of concerns: separating code into different files based on use
  • Do Not Repeat Yourself (DRY) Code: encourages devs to reuse or share pieces fo code in order to reduce the number of lines of code that need to be written or maintained
  • PX: Pixels
  • Variables are containers that store values. You start by declaring a variable with the let keyword, followed by the name you give to the variable:
  • Operator is a mathematical symbol that produces a result based on two values (or variables).
  • Conditionals are code structures used to test if an expression returns true or not

  • Array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
